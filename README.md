#  Weather Checker (OpenWeatherMap API)

A simple web page that connects to the **OpenWeatherMap Current Weather API** using JavaScript.  
The user enters a city name and the app displays real-time weather details.

##  Features
- Temperature, feels-like, min & max
- Weather description + icon
- Humidity & wind speed
- Sunrise & sunset
- Error handling for invalid cities

##  API Documentation
https://openweathermap.org/current

##  Files
- index.html  
- style.css  
- script.js  

##  How to Use
1. Enter a city name  
2. Click **Get Weather**  
3. Weather info appears on screen
